# parle-landing

Minimal Parle marketing site.

## Stack

- Astro 5
- Tailwind CSS 4
- pnpm
- Node 22 via mise

## Commands

```sh
pnpm dev
pnpm build
pnpm format
```

## Conventions

- Keep the site intentionally minimal.
- Prefer static pages and simple Astro components.
- Validate with `pnpm build` before handoff.
- No em dashes in file output.
